Weso

用 Web 技术渲染界面、为页面 JavaScript 提供原生能力的 Windows 应用开发框架

支持调用 C/C++/DLL，页面内运行 Python 脚本，最终打包成单文件 exe

下载 weso.exe 查看文档 GitHub 仓库

当前版本：…

核心能力

页面运行时 window.W（及 window.weso）已就绪，提供以下能力域

📁

文件 I/O

文本/二进制/base64 读写、目录创建与列举、文件选择器、资源包读取、文件拖拽监听

⚙️

系统

原生消息框、执行 cmd 命令、环境变量读写、捕获 stdout/stderr、退出应用

🪟

窗口管理

创建/销毁子窗口、拖拽移动、关闭拦截、系统托盘、窗口间通信、键盘/鼠标 Hook

➕

DLL 互操作

声明 proto，native 自动处理类型转换。支持按句柄、按地址调用与 Dll 封装类

🐍

Python 集成

运行时下载安装、运行脚本/文件、模块搜索路径、JS ↔ Python 双向通信

📦

CLI 打包

新建/调试/打包项目为单文件 exe，脚本加密存储，内置卡密功能

示例 Playground

内置 55 个分类可运行脚本：系统、文件、窗口、C++、Python，选中即可查看代码、运行并实时观察结果
